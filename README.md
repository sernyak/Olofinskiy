# Микола Олофінський — сайт

Персональний сайт психотерапевта **Миколи Олофінського** (психотерапевт · спеціаліст з екстремальних переговорів · ветеран ЗСУ · 20+ років практики).

## Структура

| Шлях | Що це |
|---|---|
| `/` (`index.html`) | Сайт-візитка — головна сторінка |
| `/landing/` | Конверсійний лендінг для запису на консультацію |

Чистий HTML/CSS/JS без залежностей. Шрифти — Google Fonts (Playfair Display, Lora, Manrope).

## Локальний запуск

```bash
python3 -m http.server 8000
# відкрити http://localhost:8000/
```

## Хостинг

Сайт публікується через **GitHub Pages** (гілка `main`, корінь). Файл `.nojekyll` вимикає обробку Jekyll для коректної віддачі статики.
